Este espaço é reservado exclusivamente para documentar e validar a lógica matemática de um Registrador de Deslocamento com Realimentação Linear (LFSR) de 8 bits.

Aqui você encontra o simulador algorítmico desenvolvido em Python que atua como nosso Golden Model. O objetivo deste código é emular o comportamento do hardware, rastreando o caminho percorrido pelos bits, a operação da porta XOR e a ausência de estados de travamento (lock-up) antes da etapa de implementação física do chip em silício.

🛠️ Tecnologias e Ferramentas:
Python | Verilog | SPICE / Ngspice | Magic VLSI | Xschem | ModelSim | Linux
